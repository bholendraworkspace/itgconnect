# **App Name**: ITG Connect

## Core Features:

- Secure User Authentication: Enable user signup and login through Google OAuth, with subsequent profile completion for personal details, photo, and optional department information.
- Personalized Employee Dashboard: Display upcoming, today's, and past employee birthdays, with the functionality for users to send personalized wishes to colleagues.
- Self-Managed Employee Profiles: Allow users to view and update their personal details, including profile photo, and manage their individual achievements.
- Organization-Wide Achievement Feed: A dynamic, centralized feed showcasing recent accomplishments and achievements from employees across the entire organization.
- Office News & Events Hub: A dedicated section for administrators to post and employees to browse internal events, announcements, and company activities.
- Essential Developer Tool Suite: Provide practical developer utilities such as a JSON viewer, HTML viewer, Base64 encoder/decoder, and Timestamp converter.
- AI API Response Explainer Tool: An AI-powered tool that acts as a valuable assistant to parse complex API responses, offering human-readable summaries, explanations of key fields, and potentially suggesting next steps or common issues. This tool enhances the developer utility suite by making API responses more accessible.

## Style Guidelines:

- The brand's primary visual elements will utilize a dynamic linear gradient (103deg, #FF006A 0%, #FF7A00 100%), reflecting energy and innovation.
- A very light desaturated blue-grey (#EBEEF6) serves as the background color, ensuring readability and reducing eye strain while maintaining a clean, open aesthetic.
- The 'Inter' typeface, a modern grotesque sans-serif, is recommended for all text elements. Its objective, neutral, and highly readable characteristics make it ideal for both headlines and body text within an internal application where clarity is paramount.
- Utilize a consistent set of minimalist, vector-based icons throughout the application. The clean lines and clear metaphors of these icons will support quick comprehension and maintain a sleek, modern look.
- Employ a responsive, grid-based layout that adapts seamlessly to various screen sizes and devices. This ensures an organized presentation of content and optimal usability for all internal users, regardless of their access method.
- Implement subtle and purposeful animations for user feedback, such as gentle fades on component transitions, discreet hover effects on interactive elements, and micro-interactions on data updates. These animations enhance the user experience without being distracting.
